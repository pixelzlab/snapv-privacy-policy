6. Data Storage and Security

6.1 Local Storage

  • Downloaded stock videos are stored in Movies/StockVideos folder on your device
  • App settings and preferences are stored locally using Android DataStore
  • Download queue and history are stored in local database (Room)
  • Cloud OAuth tokens (if you connect cloud storage) are stored securely on device
  • None of this data is transmitted to our servers or third parties

6.2 Data Security

We implement security measures to protect your data:

  • HTTPS Encryption: All network communications use encrypted HTTPS connections
  • No Cleartext Traffic: HTTP (unencrypted) connections are blocked
  • Local Storage: App data is protected by Android's app sandbox
  • No Account System: No user accounts or passwords to compromise

6.3 Data Retention

Data Type Retention Period
Downloaded Files Until you manually delete them
App Settings Until you uninstall the app or clear data
Analytics Data 14 months (Firebase default) - Anonymous, server-side
Crash Reports 90 days (Firebase default) - Server-side
